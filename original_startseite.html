Erinnere Dich,

Du bist mehr, als Du denkst.

In dir lebt die Erde, 

in dir ruht der Himmel. 

Zwischen Herz und Atem

zeigt sich das Ganze. 

Werde still ... 

und erinnere dich.

... reinige … weine … 
... las gehen, was gehen will ... 
... erfrische dich… 
... ströme …

Du bist FEUER


... brenne ... lodere ... sprühe ... 
... reinige ... entfalte ... verändere ...
... dehne dich aus ... verwandle ... 
... knistere ... inspiriere ...
... leuchte ... wärme … 
... tanze …

 Du bist LUFT

... flüstere … 
… lass dich tragen ... 
... fantasiere ... beobachte ... 
... fokussiere ... 
... kläre ... 
... entscheide ...

Du bist ERDE

… danke …
... schenke ... behüte ...

... achte die Fülle ...

… nähre dich ...
